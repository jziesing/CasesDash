# CasesDash
to keep track of cases

**this project is under development**

See working build [here](http://casesziesing.herokuapp.com/):

### Install

Using [npm](https://www.npmjs.com/):

    $ npm install

### Build

Using [webpack](https://webpack.github.io/):

    $ webpack

### Start

Using [npm](https://www.npmjs.com/):

    $ npm start

### Use

    http://localhost:3000/
